# Coders of Bangalore - Data Analysis Project

## 📌 Overview
This project analyzes data of coders in Bangalore to uncover trends, skills distribution, and technology usage.

## 📊 Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of coder trends

## 📈 Insights
- Popular programming languages
- Skill distribution among coders
- Trends in Bangalore tech community

## 🛠️ Technologies Used
- Pure Python
